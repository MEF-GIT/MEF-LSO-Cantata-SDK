Class: CarrierEthernetSls  (Carrier Ethernet SLS)

Data type that represents Carrier Ethernet Service Level Specification that provides a list of Performance Metrics where each item in the list includes the parameters and performance objective for the given Performance Metric. It is associated with EVC or OVC and a list of SlsCosNameEntry(s). Reference MEF 10.4 Section 8.8 EVC Service Specification Service Attribute and MEF 26.2 Section 12.13 OVC Service Level Specification Service Attribute."
